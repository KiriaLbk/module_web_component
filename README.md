# CustomeComponent

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.2.14.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Build

Run `npm run build-coponent` to build the project.

Тег компонента 
```JavaScript
<web-text></web-text>
```

Склееный файл компонента dist/custome-component/web-text-element.js

в себе содержит:
 * dist/custome-component/polyfills.js
 * dist/custome-component/runtime.js
 * dist/custome-component/main.js
